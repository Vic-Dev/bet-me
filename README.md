Sinatra
=============

## Getting Started

1. `bundle install`
2. `bundle exec rake db:drop`
2. `bundle exec rake db:create`
3. `bundle exec rake db:migrate`
4. `bundle exec rake db:seed`
5. `bundle exec shotgun -p 3000 -o 0.0.0.0`
6. Visit `http://localhost:3000/` in your browser


# BetMe
